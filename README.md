# serverspec-demo-1
serverspec-demo-1は、Serverspecを使用して対象となるAWSのEC2インスタンス上でテストを自動で行います。  
# 使用技術
- ServerSpec
  - Ruby
- AWS
  - VPC
  - EC2
  - RDS
